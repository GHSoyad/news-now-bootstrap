Difference between var, let and const keywords in JavaScript

Golam Hasnain Soyad

var keyword can be declared without initialization and it can be updated and redeclared again.

let keyword can be declared without initialization and it can be updated but cannot be redeclared.

const keyword cannot be redeclared without initialization and it cannot be updated or redeclared.

Difference between regular functions and arrow functions

Golam Hasnain Soyad

Unlike regular functions, arrow functions do not have their own this. The value of this inside an arrow function remains the same throughout the lifecycle of the function and is always bound to the value of this in the closest non-arrow parent function.

Regular functions created using function declarations or expressions are constructible and callable However, the arrow functions are only callable and not constructible

Difference between map, forEach, find and filter methods

Golam Hasnain Soyad

The map() method creates an entirely new array. The map() method returns the newly created array according to the provided callback function.

The forEach() method does not create a new array based on the given array. The forEach() method returns “undefined“.

The find() method returns the first value that matches from the collection. Once it matches the value in findings, it will not check the remaining values in the array collection.

The filter() method returns the matched values in an array from the collection. It will check all values in the collection and return the matched values in an array.

Template strings lets us insert/interpolate variables and expressions into strings without needing to concatenate like in older versions of JavaScript.

Template strings allows us to create strings that are complex and contain dynamic elements.

We can use template strings to write multiple strings easily.

Using template strings makes code readability efficient and easier.
